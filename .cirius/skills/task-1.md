# Task 1: Configurar Projeto React e Tailwind

**Skill:** code
**Brain Type:** code

## Prompt

Crie um novo projeto React usando Vite. Configure o Tailwind CSS seguindo a documentação oficial, incluindo a criação dos arquivos `tailwind.config.js` e `postcss.config.js`. Adicione as diretivas `@tailwind base; @tailwind components; @tailwind utilities;` ao arquivo `src/index.css`. Verifique se o Tailwind está funcionando corretamente adicionando uma classe de texto no `App.tsx`.
