# Task 3: Criar Componente de Landing Page

**Skill:** code
**Brain Type:** code

## Prompt

Crie um novo componente React chamado `LandingPage.tsx` dentro da pasta `src/components`. Este componente deve renderizar um elemento `div` centralizado na tela, com a classe `flex items-center justify-center min-h-screen`. Dentro deste `div`, adicione um elemento `h1` com o texto 'Olá Mundo!' e a classe `text-5xl font-bold text-gray-900 dark:text-gray-50` para garantir que o texto seja grande e visível, utilizando as cores do Tailwind.
