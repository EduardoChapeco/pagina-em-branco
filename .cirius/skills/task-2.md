# Task 2: Instalar e Configurar shadcn/ui

**Skill:** code
**Brain Type:** code

## Prompt

Instale e configure o shadcn/ui no projeto React. Siga as instruções de instalação para React e Tailwind. Inicialize o shadcn/ui usando o comando `npx shadcn-ui@latest init`. Certifique-se de que o `components.json` seja criado e que as variáveis CSS personalizadas sejam adicionadas ao `src/index.css`.
